# Integration Engineering Plugin

Claude Code integration skills — Dispatch, Remote Control, Channels, Slack, and Scheduled Tasks. Work away from your terminal.

## Skills

| Skill | Description |
|-------|-------------|
| `dispatch` | Send tasks from your phone via the Claude mobile app to Desktop sessions |
| `remote-control` | Drive running CLI/VS Code sessions from claude.ai/code or mobile |
| `channels` | Push events from Telegram, Discord, iMessage, or webhooks into sessions |
| `slack-integration` | @Claude mentions in Slack channels spawn Claude Code web sessions |
| `scheduled-tasks` | /loop, CronCreate, Desktop, and cloud scheduled tasks on recurring schedules |
