# Data Engineering Plugin

Package management, AST indexing, scheduled tasks, and Supabase with Cube.js semantic layer.

## Commands

| Command | Description |
|---------|-------------|
| `/packages` | Audit, update, and manage packages and pinned versions across the monorepo |
| `/ast-index` | Rebuild the L1-L4 tree-sitter AST index for semantic search |
| `/schedule` | Create, list, update, or delete Claude Code scheduled tasks and cron jobs |
| `/cube-model` | Define or update Cube.js semantic layer models backed by Supabase |

## Skills

| Skill | Description |
|-------|-------------|
| `package-manager` | Manage npm/pip/cargo packages, pin versions, audit drift, bulk update |
| `ast-index` | Rebuild tree-sitter L1-L4 index (files, symbols, signatures, dependencies) |
| `scheduled-tasks` | Manage Claude Code remote triggers, cron schedules, and recurring agents |
| `supabase-cube` | Setup and manage Supabase tables with Cube.js semantic layer definitions |
