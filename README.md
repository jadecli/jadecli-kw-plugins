# jadecli-kw-plugins

Knowledge-worker plugins for Claude Code and Cowork by jadecli.

## Installation

```bash
# Add the marketplace
claude plugin marketplace add jadecli/jadecli-kw-plugins

# Install a plugin
claude plugin install data-engineering@jadecli-kw-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `data-engineering` | Package management, AST indexing, scheduled tasks, Supabase + Cube.js semantic layer |
